## 🎨 Cool Features

### 🌠 Animated Starfield Background
- Smooth canvas animation  
- Respects your system's "reduce motion" setting  
- Lightweight & performant  

### 💬 Rotating Motivational Quotes
- Fresh quote every few seconds  
- Pause on hover  
- Fade animations  

### 🎮 Rock Paper Scissors Game
- Play right in the browser  
- Score saved locally (won't reset on refresh)  
- Accessible & keyboard-friendly  

### 📱 Fully Responsive
Looks great on phone, tablet & desktop  
No horizontal scrolling, ever

## 🛠️ Tech Stack

✅ HTML5  
✅ CSS3 (Flexbox, Grid, Variables)  
✅ Vanilla JavaScript (ES6+)  
✅ Canvas API (for starfield)  
✅ LocalStorage API (for game scores)  
✅ Google Fonts (Poppins)  

No frameworks. No build tools. Just clean code.
